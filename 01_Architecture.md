# Architecture

## Summary

The system is organized into ten layers: development, model, knowledge, RAG, agent, cybersecurity lab, automation, governance, portfolio, and dashboard.

## Architecture Rule

Agents recommend, draft, classify, summarize, and structure. Humans approve destructive, external, legal, financial, or security-sensitive actions.

## STICHES Integration

`DESIGN.md` controls the documentation and future UI identity. Any dashboard, report, or agent-generated artifact should follow its color, typography, layout, and governance rules.
