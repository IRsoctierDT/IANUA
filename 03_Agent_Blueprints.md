# Agent Blueprints

## SOC Analyst Agent

Analyzes logs and alerts. Produces severity, event type, indicators, recommended actions, assumptions, and evidence.

## Threat Intelligence Agent

Summarizes indicators, suspicious domains, threat reports, and confidence levels.

## Vulnerability Assessment Agent

Converts scan results into remediation priorities.

## Legal/Compliance Research Agent

Supports issue analysis, authority review, citation checklists, and draft preparation. It does not replace legal counsel.

## Business Proposal Agent

Converts client needs into structured proposals and scopes of work.

## Knowledge Curator Agent

Organizes notes, transcripts, PDFs, and research into retrieval-ready knowledge.

## Portfolio Documentation Agent

Turns lab work into GitHub-ready README files, reports, and case studies.

## Executive Assistant Agent

Supports planning, prioritization, review cycles, and decision logs.
