# Streamlit SOC Dashboard

## Purpose

The Streamlit dashboard provides a local web interface for IANUA.

## Capabilities

- Paste a security log entry
- Run SOC workflow
- Display SOC analysis
- Display MITRE ATT&CK mapping
- Display threat intelligence classification
- Preview Markdown report
- Download PDF incident report

## Run Command

streamlit run dashboard/app.py

## Current Status

Prototype dashboard operational.
