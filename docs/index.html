Get started

Up in three steps

Python 3.11+ and Ollama are the only prerequisites.

1
Clone & install 
# clone the repo
git clone https://github.com/\
  IRsoctierDT/\
  ai-operator-cyber-command-center

cd ai-operator-cyber-command-center
pip install -e ".[dev]"
2
Configure & run CI 
# copy environment template
cp .env.example .env

# run full quality gate locally
python -m pytest
ruff check .
mypy agents scripts tests
bandit -r agents scripts
3
Run the pipeline 
# analyse a plain-text log
python scripts/run_pipeline.py \
  --log "Failed password for root \
  from 203.0.113.42 port 22"

# or a structured JSON event
python scripts/run_pipeline.py \
  --file sample-logs/\
  structured_events.json
