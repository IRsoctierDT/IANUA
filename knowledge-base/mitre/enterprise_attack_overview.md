# MITRE ATT&CK Enterprise Overview

MITRE ATT&CK is a knowledge base of adversary tactics, techniques, and procedures based on real-world observations.

## Core Use

- Map observed behavior to adversary techniques
- Support detection engineering
- Support incident investigation
- Standardize defensive reporting
