# SOC Fundamentals

A Security Operations Center monitors, detects, investigates, and responds to cybersecurity events.

## Core SOC Duties

- Monitor logs and alerts
- Investigate suspicious activity
- Correlate events across systems
- Preserve evidence
- Escalate high-risk incidents
- Document findings

## Common Evidence Sources

- Firewall logs
- IDS/IPS alerts
- Authentication logs
- Endpoint alerts
- DNS logs
- Packet captures

## Analysis Rule

Separate facts, assumptions, analysis, recommendations, and unknowns.
